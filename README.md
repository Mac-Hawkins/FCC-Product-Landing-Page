# Product Landing Page - freeCodeCamp Responsive Web Design

This project is a simple **Product Landing Page** I made as a part of completing freecodecamp's _Responsive Web Design_ certification.

In order to complete the project, I had to implement a list of items which you can see me refer to in the comments throughout the code. While the project had many requirements, the main tasks were the following:

- Implement a navbar that would take you to different sections within the page
- Display an image within the #header element such as a company logo
- Embed a video
- Implement a form for users to enter their email and submit, which would then send an email to a mock email address
  - I have since disabled the sending to the mock email address and now simply disable the submit button and show a message on a successful submit

If you wish to see more projects I completed for the course, you can view each one at the bottom of [my certificate page](https://www.freecodecamp.org/certification/mac8/responsive-web-design#:~:text=responsive%2Dweb%2Ddesign-,As,-part%20of%20this).

## About This Version

The course had you write your code in CodePen, which behaves slightly differently as it automatically links the HTML and CSS together and doesn't allow the uploading of images without a subscription. As such, I made some modifications from the original code, which included the following:

- Extracted SVG code into its own file
- Added the link for the external CSS
- Added meta, body, and html tags
- Removed a lot of the preset freeCodeCamp comments
- Formatted files
